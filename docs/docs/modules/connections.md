# 连接数
连接模块支持连接到各种结构化、半结构化和非结构化数据存储引擎。将多维数据纳入框架，实现自然语言与多维数据的交互

我们目前支持的数据源列表如下。

|数据源|支持|笔记|
| ------------------------------------------------------------------------------------------ |---------| ------------------------------------------- |
| [MySQL](https://www.mysql.com/) |是的 |  MySQL 是世界上最流行的开源数据库。                             |
| [PostgresSQL](https://www.postgresql.org/) |是的 |  全球最先进的开源关系数据库|
| [Vertica](https://www.vertica.com/) |是的 |  Vertica 是一个高度一致、符合 ACID 的 SQL 数据仓库，专为当今数据驱动世界的规模和复杂性而构建。                                   |
| [Spark](https://github.com/apache/spark) |是的 |  用于大规模数据分析的统一引擎 |
| [DuckDB](https://github.com/duckdb/duckdb) |是的 |  DuckDB是一个进程内SQL OLAP数据库管理系统|
| [Sqlite](https://github.com/sqlite/sqlite) |是的 |                                             |
| [MSSQL](https://github.com/microsoft/mssql-jdbc) |是的 |                                             |
| [ClickHouse](https://github.com/ClickHouse/ClickHouse) |是的 |  ClickHouse 是用于实时应用程序和分析的速度最快、资源效率最高的开源数据库。                                      |
| [Oracle](https://github.com/oracle) |没有 |           待办事项 |
| [Redis](https://github.com/redis/redis) |没有 |  多模型NoSQL数据库|
| [MongoDB](https://github.com/mongodb/mongo) |没有 |  MongoDB 是一个开源的跨平台面向文档的数据库程序 |
| [HBase](https://github.com/apache/hbase) |没有 |  开源、分布式、版本化、面向列的存储建模 |
| [多丽丝](https://github.com/apache/doris) |是的 |  Apache Doris 是一个易于使用、高性能和统一的分析数据库。                              |
| [DB2](https://github.com/IBM/Db2) |没有 |           待办事项 |
| [Couchbase](https://github.com/couchbase) |没有 |           待办事项 |
| [Elasticsearch](https://github.com/elastic/elasticsearch) |没有 |  免费开放、分布式、RESTful 搜索引擎 |
| [OceanBase](https://github.com/OceanBase) |没有 |  OceanBase是一个分布式关系数据库。                               |
| [TiDB](https://github.com/pingcap/tidb) |没有 |           待办事项 |
| [StarRocks](https://github.com/StarRocks/starrocks) |是的 | StarRocks 是下一代高性能分析数据仓库 |
