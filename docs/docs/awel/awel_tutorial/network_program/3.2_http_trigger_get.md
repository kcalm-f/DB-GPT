# 3.2 处理GET请求

在上一节中，我们创建了一个返回固定字符串的简单 HTTP 触发器。在 
在本节中，我们将创建一个新的 HTTP 触发器，它根据以下内容返回一个字符串： 
请求的查询参数。

## 向某人问好

在开始编写代码之前，我们需要在您的计算机中安装“pydantic”包 
项目 [awel-tutorial](/docs/awel/awel_tutorial/getting_started/1.1_hello_world#creating-a-projec)
```bash
poetry add "pydantic>=2.6.0"
```
然后在`awel_tutorial`目录中创建一个名为`http_trigger_say_hello.py`的新文件并添加以下代码：
```python
from dbgpt._private.pydantic import BaseModel, Field
from dbgpt.core.awel import DAG, HttpTrigger, MapOperator, setup_dev_environment

class TriggerReqBody(BaseModel):
    name: str = Field(..., description="User name")
    age: int = Field(18, description="User age")

with DAG("awel_say_hello") as dag:
    trigger_task = HttpTrigger(
        endpoint="/awel_tutorial/say_hello", 
        methods="GET", 
        request_body=TriggerReqBody,
        status_code=200
    )
    task = MapOperator(
        map_function=lambda x: f"Hello, {x.name}! You are {x.age} years old."
    )
    trigger_task >> task

setup_dev_environment([dag], port=5555)
```
并运行以下命令来执行代码：
```bash
poetry run python awel_tutorial/http_trigger_say_hello.py
```
现在，打开一个新终端并运行以下命令向服务器发送 GET 请求：
```bash
curl -X GET \
"http://127.0.0.1:5555/api/v1/awel/trigger/awel_tutorial/say_hello?name=John&age=25"
```
输出应如下所示：
```plaintext
"Hello, John! You are 25 years old."
```
然后你可以通过按“Ctrl+C”来停止服务器。

在上面的代码中，我们创建了一个继承自 BaseModel 的 TriggerReqBody 类 
定义请求体的结构。模块 `dbgpt._private.pydantic` 包装 
pydantic 用于兼容不同版本的 pydantic。

当收到请求时，http触发器会解析查询参数并构建一个 
`TriggerReqBody` 对象，然后将其传递给下一个操作员。

## 返回 JSON 响应

在上一节中，我们返回一个字符串作为响应。我们还可以返回一个 
JSON 响应。

AWEL使用`fastapi`作为默认的Web框架，它会自动判断
任务输出的响应类型，常见的`dict`、`list`、`pydantic`的`BaseModel`， 
等将自动转换为 JSON 响应。

因此，如果你想返回 JSON 响应，你可以简单地返回一个 `dict` 或一个 `BaseModel`。

在`awel_tutorial`目录中创建一个名为`http_trigger_say_hello_json.py`的新文件，并添加以下代码：
```python
from dbgpt._private.pydantic import BaseModel, Field
from dbgpt.core.awel import DAG, HttpTrigger, MapOperator, setup_dev_environment

class TriggerReqBody(BaseModel):
    name: str = Field(..., description="User name")
    age: int = Field(18, description="User age")

with DAG("awel_say_hello_json") as dag:
    trigger_task = HttpTrigger(
        endpoint="/awel_tutorial/say_hello_json", 
        methods="GET", 
        request_body=TriggerReqBody,
    )
    task = MapOperator(
        map_function=lambda x: {"message": f"Hello, {x.name}! You are {x.age} years old."}
    )
    trigger_task >> task

setup_dev_environment([dag], port=5555)
```
并运行以下命令来执行代码：
```bash
poetry run python awel_tutorial/http_trigger_say_hello_json.py
```
现在，打开一个新终端并运行以下命令向服务器发送 GET 请求：
```bash
curl -X GET \
"http://127.0.0.1:5555/api/v1/awel/trigger/awel_tutorial/say_hello_json?name=John&age=25"
```
您将得到以下输出：
```plaintext
{"message":"Hello, John! You are 25 years old."}
```
然后你可以通过按“Ctrl+C”来停止服务器。
