# 1.1 你好世界

## 准备工作

在本教程中，我们将使用“poetry”来管理我们的项目依赖项。如果您没有安装 `poetry`，您可以按照[此处](https://python-poetry.org/docs/) 的说明进行安装。

## 创建项目

您将首先创建一个新的 python 项目。你可以随意命名它；对于本教程，我们将其称为“awel-tutorial”。

我们建议在您的主目录中创建一个项目目录，但您可以将其放在您喜欢的任何位置。

打开终端并运行以下命令来创建项目目录和 AWEL 教程目录：

对于 Linux、macOS 或 PowerShell，请输入：
```bash
mkdir -p ~/projects
cd ~/projects
```
然后，运行以下命令创建一个新项目并更改到新目录：
```bash
poetry new awel-tutorial
cd awel-tutorial
```
项目的树应该如下所示：
```plaintext
awel-tutorial
├── README.md
├── awel_tutorial
│   └── __init__.py
├── pyproject.toml
└── tests
    └── __init__.py
```
## 添加 DB-GPT 依赖项
```bash
poetry add "dbgpt>=0.5.1"
```
## 首先你好世界

接下来，您将创建一个简单的 DAG，将“Hello, world”打印到控制台。

现在在`awel_tutorial`目录中创建一个名为`first_hello_world.py`的新文件并添加以下代码：
```python
from dbgpt.core.awel import DAG, MapOperator

with DAG("awel_hello_world") as dag:
    task = MapOperator(map_function=lambda x: print(f"Hello, {x}!"))
task._blocking_call(call_data="world")
```
现在，项目的树应该如下所示：
```plaintext
awel-tutorial
├── README.md
├── awel_tutorial
│   ├── __init__.py
│   └── first_hello_world.py
├── poetry.lock
├── pyproject.toml
└── tests
    └── __init__.py

```
然后，运行以下命令来执行代码：
```bash
poetry run python awel_tutorial/first_hello_world.py
```
你会看到“你好，世界！”打印到控制台。
```bash
Hello, world!
```
## AWEL 代码剖析

让我们分解一下您刚刚编写的代码。
```python
with DAG("awel_hello_world") as dag:
    task = MapOperator(map_function=lambda x: print(f"Hello, {x}!"))
```
此代码创建一个名为“awel_hello_world”的新 DAG（有向无环图）。 
“MapOperator”是一个简单的运算符，它接受一个函数并使用数据调用它 
传递给它。在本例中，该函数是一个 lambda，它打印“Hello, world!”到控制台。


该任务是“MapOperator”类的实例。我们调用“call”方法 
将“call_data”参数设置为“world”的任务。
```python
task._blocking_call(call_data="world")
```
当您调用任务时，将使用您传递给它的 data(`"world"`) 来调用 lambda 函数。

`_blocking_call` 方法用于以阻塞方式调用任务。只为 
这里进行测试，下一节我们会找到更好的调用任务的方法。


## 使用 `asyncio` 你好世界

AWEL 中的所有任务调用都是异步的。此示例展示了如何运行任务
异步。

在`awel_tutorial`目录中创建一个名为`first_hello_world_asyncio.py`的新文件，并添加以下代码：
```python
import asyncio

from dbgpt.core.awel import DAG, MapOperator

with DAG("awel_hello_world") as dag:
    task = MapOperator(map_function=lambda x: print(f"Hello, {x}!"))

asyncio.run(task.call(call_data="world"))
```
并运行以下命令来执行代码：
```bash
poetry run python awel_tutorial/first_hello_world_asyncio.py
```
你会看到“你好，世界！”打印到控制台。
```bash
Hello, world!
```
## Hello World 有两个任务

当我们调用单个节点时，我们可以向它传递数据。这个例子展示了如何传递数据
使用 InputOperator 执行任务。

在“awel_tutorial”中创建一个名为“first_hello_world_two_tasks.py”的新文件 
目录并添加以下代码：
```python
import asyncio

from dbgpt.core.awel import DAG, MapOperator, InputOperator, SimpleCallDataInputSource

with DAG("awel_hello_world") as dag:
    input_task = InputOperator(
        input_source=SimpleCallDataInputSource()
    )
    task = MapOperator(map_function=lambda x: print(f"Hello, {x}!"))
    input_task >> task
    

asyncio.run(task.call(call_data="world"))
```
并运行以下命令来执行代码：
```bash
poetry run python awel_tutorial/first_hello_world_two_tasks.py
```
你会看到“你好，世界！”打印到控制台。
```bash
Hello, world!
```
在这种情况下，我们有两个任务。第一个任务是一个获取数据的“InputOperator” 
来自“SimpleCallDataInputSource”。第二个任务是一个“MapOperator”，它接受 
来自第一个任务的数据并打印“Hello, world!”到控制台。

我们使用“>>”运算符来连接两个任务。该运算符用于定义 
任务之间的父子关系，也称为任务依赖关系。
您也可以使用 set_downstream 方法定义任务依赖关系，示例如下：
```python
input_task.set_downstream(task)
```
上面的单任务 DAG 是两个任务 DAG 的一种特例，其中未使用“InputOperator”。
```python
with DAG("awel_hello_world") as dag:
    task = MapOperator(map_function=lambda x: print(f"Hello, {x}!"))

asyncio.run(task.call(call_data="world"))
```
## DAG 可视化

安装 graphviz 包以可视化 DAG 图。
```bash
poetry add graphviz
```
修改`first_hello_world_two_tasks.py`文件添加以下代码：
```python
dag.visualize_dag()
```
完整的代码是这样的：
```python
import asyncio

from dbgpt.core.awel import DAG, MapOperator, InputOperator, SimpleCallDataInputSource

with DAG("awel_hello_world") as dag:
    input_task = InputOperator(
        input_source=SimpleCallDataInputSource()
    )
    task = MapOperator(map_function=lambda x: print(f"Hello, {x}!"))
    input_task >> task

dag.visualize_dag()
asyncio.run(task.call(call_data="world"))
```
再次运行“first_hello_world_two_tasks.py”：
```bash
poetry run python awel_tutorial/first_hello_world_two_tasks.py
```
您将看到以下输出：
```bash
InputOperator(node_id=a307d921-3bd0-423d-80f0-30aa25aaa9fe)
 -> MapOperator(node_id=bdb335f8-179d-4e08-b1ec-3b58a52d1e84)
Hello, world!
```
