# 数据源

DB-GPT数据源模块旨在管理企业的结构化和半结构化数据资产，将数据库、数据仓库、数据湖等连接到DB-GPT框架，快速构建基于数据的智能应用和大模型。目前，DB-GPT支持一些常见的数据源，也支持自定义扩展。

您可以通过右上角的**添加数据源**按钮来添加数据源。在弹出的对话框中选择对应的数据库类型并填写所需参数即可完成添加。

## 支持的数据源类型

当前文档涵盖或正在扩展以下数据源：

- MySQL
- SQLite
- ClickHouse
- PostgreSQL
- 鸭数据库
- 蜂巢
- 微软SQL
- 甲骨文
- OceanBase
- 高斯数据库
- 开放高斯
- 阿帕奇·多丽丝
- 星石
- 维蒂卡
