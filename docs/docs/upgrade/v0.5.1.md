# 升级到v0.5.1

## 概述

本指南适用于从 v0.5.0 升级到 v0.5.1。如果你使用 SQLite，
您不需要升级数据库。如果使用MySQL，则需要升级数据库。

## 准备

### 备份您的数据库

为防止数据丢失，建议升级前备份数据库。
备份方式根据您的数据库类型而定。

## 升级

### 停止 DB-GPT 服务

根据您的启动方式停止DB-GPT服务。

### 升级数据库

执行以下SQL来升级数据库。

**添加列**
```sql
USE dbgpt;
ALTER TABLE  dbgpt_serve_flow
    ADD COLUMN `error_message` varchar(512) null comment 'Error message' after `state`;
```
